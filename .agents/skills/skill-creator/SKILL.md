# Skill Creator Instructions

Когда пользователь просит создать новый "навык" (скилл) или агента, вы должны выполнить следующие действия:
1. Создайте дочернюю папку для скилла по пути: `.agents/skills/<название_скилла>/` в корне проекта.
2. Внутри создайте файл `SKILL.md` с обязательной YAML-шапкой (name, description).
3. В теле файла `SKILL.md` подробно опишите инструкции и правила для вас (AI-агента) о том, как нужно будет выполнять этот скилл в будущем.
4. Убедитесь, что инструкции предельно точные, так как при использовании скилла базовый контекст будет дополняться этими правилами.
